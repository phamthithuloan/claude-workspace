## Hướng dẫn paste stock images vào bài

1. Mở draft file hoặc Google Doc.
2. Tìm anchor bằng CMD/CTRL+F với snippet ~30 ký tự ở cột "Anchor".
3. Chèn ảnh stock: Insert → Image → By URL → paste URL Pexels.
4. Paste caption ngay dưới ảnh, **italic + căn giữa**.
5. Ảnh stock: KHÔNG chèn logo Hồng Ngọc.
